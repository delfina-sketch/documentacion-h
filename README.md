# Manual Hermes

En este manual se explicarán los conceptos básicos para el uso del administrador de Hermes.

Hermes es el panel de administración multi-tenant para e-commerce desarrollado por POW. Permite gestionar todos los aspectos de una tienda online: productos, pedidos, clientes, pagos, envíos, marketing, contenidos y más.

## Acceso al panel

Se accede al panel desde el siguiente enlace: [https://hermes.pow.la/](https://hermes.pow.la/)&#x20;

Al ingresar al mismo, se puede navegar desde el menú lateral a los diferentes módulos. También se pueden buscar módulos específicos utilizando la barra de búsqueda superior.

En los casos donde se administre más de una tienda, se podrá ingresar de una a otra desde la esquina superior derecha.&#x20;

## Secciones principales

| Seccion            | Descripcion                                                            |
| ------------------ | ---------------------------------------------------------------------- |
| **Dashboard**      | Métricas y KPIs del negocio: facturación, pedidos, productos, clientes |
| **Pedidos**        | Gestión de órdenes completas y abandonadas                             |
| **Catálogo**       | Productos, prototipos, propiedades e importación masiva de productos   |
| **Categorías**     | Configuración de categorías                                            |
| **Pagos**          | Métodos de pago y promociones bancarias                                |
| **Promociones**    | Descuentos automáticos y cupones                                       |
| **Clientes**       | Base de clientes con segmentación                                      |
| **Marketing**      | XML Feeds, atención al cliente, suscriptos                             |
| **Notificaciones** | Emails transaccionales y plantillas WhatsApp                           |
| **Locales**        | Puntos de venta físicos                                                |
| **Tienda**         | Menú, contenidos, landings                                             |
| **Configuración**  | Ajustes generales, logística, usuarios, facturación                    |
| **Apps**           | App Store e integraciones instaladas                                   |
