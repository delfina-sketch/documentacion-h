# Pedidos

## Descripción&#x20;

Este módulo muestra información de todos los pedidos, ya sea de sitios web y/o marketplaces. Mediante el mismo se podrán realizar distintas gestiones (cambios, devoluciones, aprobar órdenes manualmente, entre otros).&#x20;

Se accede desde el menu lateral "Pedidos".

## Tarjetas de resumen (comunes)

Ambas vistas (completos y abandonados) muestran las mismas 4 tarjetas de resumen en la parte superior:

| Metrica              | Icono | Color   |
| -------------------- | ----- | ------- |
| **Total pedidos**    | Caja  | Azul    |
| **Completadas**      | Check | Verde   |
| **Pendientes**       | Reloj | Naranja |
| **Ingresos totales** | $     | Violeta |

## Acciones globales

* **Exportar:** Descarga del listado en formato CSV/Excel
* **Nuevo Pedido:** Crear un pedido manual (abre formulario de creacion)
