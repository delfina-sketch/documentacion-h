# Administrador de Contenidos

**URL:** `/admin/content-manager`

En este sub-modulo podrás encontrar información de configuración de contenidos visuales para la Home de tu sitio.

Los bloques de contenido de la home, son 100% autoadministrables. El administrador del sitio podrá crear distintos tipos/formatos de bloques y combinarlos según la necesidad del momento

## Acceso

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-11 a las 17.26.17.png" alt=""><figcaption></figcaption></figure>

Al ingresar al módulo, se visualizará:&#x20;

* Bloques visibles y no visibles creados, divididos en 3 secciones:&#x20;
  * Pre-header: es un encabezado con información destacada. Se utiliza principalmente para comunicar beneficios disponibles.&#x20;
  * Body: es el cuerpo de la página, donde se pueden configurar los diferentes bloques como banners, carousel de productos, entre otros.&#x20;
  * Footer: es el pie de página, que contiene links relevantes y facilitadores de información.&#x20;

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-11 a las 17.28.31.png" alt=""><figcaption></figcaption></figure>

También se mostrarán las configuraciones de aplicaciones ad hoc (es decir, que requieren maquetado adicional):&#x20;

* Whatsapp: permite autoadministrar la posición del widget y configurar el teléfono de contacto.
* Pop-up newsletter: permite editar el título, subtítulo e imagen del pop up del newsletter.
* Página 404: permite modificar la imagen de error 404.&#x20;
* Spotlight: permite crear un desplegable con banners promocionales.&#x20;

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-11 a las 17.33.28.png" alt=""><figcaption></figcaption></figure>

También se mostrará una vista previa con el contenido de la home actual, que se actualiza a medida que se hacen modificaciones en los bloques. Adicionalmente se podrá cambiar la versión con iconos de dispositivo: **Desktop**, **Tablet/Mobile**, **Pantalla completa**

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-11 a las 17.37.44.png" alt=""><figcaption></figcaption></figure>
