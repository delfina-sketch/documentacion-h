# Página 404

Esta herramienta permite personalizar la imagen de error 404 de la web, cargando versiones específicas para desktop y mobile.

<figure><img src="../../../.gitbook/assets/image (8).png" alt=""><figcaption></figcaption></figure>

### Configuración

1. Ir a Tienda > Administrador de contenidos > Página 404:

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-12 a las 12.58.35.png" alt=""><figcaption></figcaption></figure>

2. En el panel lateral, completar los campos solicitados:&#x20;

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-12 a las 13.00.14.png" alt=""><figcaption></figcaption></figure>

* Imagen destkop: No debe superar 1MB de peso. Se recomienda usar formatos livianos como jpg, jpeg o png.
* Imagen mobile: No debe superar 1MB de peso. Se recomienda usar formatos livianos como jpg, jpeg o png.

{% hint style="info" %}
Consideraciones generales:

* Si se elimina una imagen cargada, se reemplazará automáticamente por la imagen default definida para la tienda.
* Para poder utilizar esta funcionalidad, se debe **solicitar previamente el maquetado** correspondiente a través del [formulario](https://form.asana.com/?k=v_sPm4Y17NxYHDezOEKt8A\&d=66032875349265) de requerimientos.
* Clientes con **Main Store + Outlet** verán configuraciones separadas por tienda. Quienes tengan solo Main, verán únicamente esa opción.
{% endhint %}

3. Guardar los cambios.
