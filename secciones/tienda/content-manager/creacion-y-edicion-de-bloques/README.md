# Creación y edición de bloques

Para crear nuevos bloques de la home, hacer click en "Nueva sección":

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-03-11 a las 17.41.33.png" alt=""><figcaption></figcaption></figure>

Es posible configurar los siguientes bloques, siempre que se encuentren maquetados:

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-03-11 a las 17.42.22.png" alt=""><figcaption></figcaption></figure>

En los siguientes módulos se explicará detalladamente como configurar cada bloque.&#x20;

Para que el bloque se visualice en el front, una vez creado se debe activar clickeando en el siguiente ícono:

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-03-11 a las 18.23.02.png" alt=""><figcaption></figcaption></figure>

Para eliminar un bloque, posarse sobre el bloque y clickear en el ícono de eliminar:&#x20;
