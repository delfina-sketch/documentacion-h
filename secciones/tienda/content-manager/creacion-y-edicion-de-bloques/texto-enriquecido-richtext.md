# Texto enriquecido (richtext)

Este bloque permite mostrar contenido en diferentes formatos y se configura mediante un código HTML otorgado por Pow.&#x20;

{% hint style="info" %}
Se deberá tener conocimientos básicos de HTML para poder autoadministrar el bloque.
{% endhint %}

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-03-12 a las 11.23.43.png" alt=""><figcaption></figcaption></figure>

### Configuración

1. En "Agregar sección" elegir la opción Texto enriquecido:

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-03-12 a las 12.05.28.png" alt=""><figcaption></figcaption></figure>

2. En el panel lateral, completar los campos solicitados:&#x20;

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-03-12 a las 12.06.04.png" alt=""><figcaption></figcaption></figure>

* **Nombre interno:** Es de uso interno para identificar y organizar los bloques más fácilmente.
* **Ancho:** colocar siempre 100% a menos que desde Pow se haya aclarado otro porcentaje.
* **Contenido HTML:** se debe cargar el código html compartido por Pow. con las modificaciones correspondientes.&#x20;
* **Programación:** una vez cargados todos los productos, se deben configurar los días y horarios de inicio y fin de visualización del banner.

3\. Luego de configurar todo, guardar los cambios. El bloque se guardará con estado oculto, y se deberá activar su visualización clickeando en el ícono de ojo:&#x20;

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-03-12 a las 12.09.16.png" alt=""><figcaption></figcaption></figure>
