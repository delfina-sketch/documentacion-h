# Menú

**URL:** `/admin/menus`

Desde este módulo se gestiona qué categorías aparecen en el menú principal de la tienda, así como ordenarlas y controlar su visibilidad.

### Acceso

Desde el menú lateral > Tienda > Menú:

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-11 a las 16.20.15.png" alt=""><figcaption></figcaption></figure>

En la pantalla principal se mostrará un buscador y el listado de categorías y subcategorías visibles.&#x20;

Dentro de cada categoría, el ícono de ojo verde indica que la categoría es visible, para ocultarla del menú se debe clickear en el mismo.&#x20;

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-11 a las 17.01.51.png" alt=""><figcaption></figcaption></figure>

Se pueden realizar dos acciones adicionales:&#x20;

* Ordenar categorías
* Configurar subcategorías dentro de la categoría
* Configurar la categoría
