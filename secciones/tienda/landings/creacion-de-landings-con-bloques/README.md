# Creación de landings con bloques

Las landings de este módulo se crean con los mismos bloques que los de la home. Se puede solicitar el diseño o bien autoadministrarlo de acuerdo a preferencia.

1. Ingresar a Tienda > Landings > Crear nueva landing:

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-03-12 a las 13.28.37.png" alt=""><figcaption></figcaption></figure>

2. Configurar el nombre de la landing. Este nombre es de uso interno:&#x20;

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-03-12 a las 13.29.37.png" alt=""><figcaption></figcaption></figure>

3. Ir a "Agregar sección" para elegir un bloque:

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-03-12 a las 13.30.36.png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-03-12 a las 13.31.37.png" alt=""><figcaption></figcaption></figure>

4. En el panel lateral, completar los campos solicitados según el bloque elegido (ver los siguientes módulos para más detalles).&#x20;
5. Guardar los cambios.&#x20;
