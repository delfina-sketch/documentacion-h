# Landings

**URL:** `/admin/landings`

Este módulo permite crear y gestionar landing pages personalizadas con editor de bloques.

### Acceso

Desde el menú lateral > Tienda > Landings:

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-12 a las 13.24.18.png" alt=""><figcaption></figcaption></figure>

Al ingresar al módulo, se mostrarán 4 tarjetas de información y un buscador:

| Metrica            | Descripcion                   |
| ------------------ | ----------------------------- |
| **Total Landings** | Total de landings creadas     |
| **Publicadas**     | Landings activas y publicadas |
| **Borradores**     | Landings en estado borrador   |
| **Total Bloques**  | Total de bloques de contenido |

También se listarán las landings ya creadas como una tarjeta con:

* Preview visual&#x20;
* Badge de estado: **Publicado** (verde) / **Borrador** (gris)
* Nombre de la landing
* Cantidad de bloques y tiempo desde ultima actualización
* Fecha de creación
* Menú de acciones: Editar, Publicar, Eliminar

Desde esta pantalla también se podrá crear nuevas landings clickeando en "Crear nueva landing".&#x20;
