# Contenidos Estaticos

**URL:** `/admin/contenidos-estaticos`

En este módulo se podrán crear y gestionar páginas estaticas como FAQ, Políticas, Términos y Condiciones, etc.

Los contenidos estáticos pueden ser de texto, con imágenes o ambas. Algunos ejemplos debajo:

* Preguntas Frecuentes
* Términos y Condiciones
* "Nuestra Historia" (información de una marca)
* Campaña Invierno 2024 (imágenes y videos de campaña)

### Acceso <a href="#acceso" id="acceso"></a>

1. Ingresar a Tienda > Contenidos estáticos:&#x20;

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-12 a las 13.04.49.png" alt=""><figcaption></figcaption></figure>

En la pantalla principal se listan los **contenidos estáticos ya creados**, junto con los botones de acción para **editar, ver y eliminar** cada uno.

Además, la pantalla cuenta con un **buscador y filtros** que permiten encontrar contenidos estáticos de forma más rápida. También se muestran **cards informativas** con datos relevantes, como el **total de contenidos creados**, la **cantidad de contenidos visibles en el header** y el **total de secciones**.

Desde esta misma pantalla también es posible **crear nuevos contenidos estáticos**.

## Columnas de la tabla

| Columna  | Descripcion                                                                     |
| -------- | ------------------------------------------------------------------------------- |
| ID       | Numero identificador (#)                                                        |
| Sección  | Nombre de la seccion (ej: info-envio, Test Video, empresa, faq, terms, privacy) |
| Titulo   | Título de la pagina                                                             |
| Path     | URL relativa de la pagina (link clickeable, ej: /info-envios, /empresa, /faq)   |
| Creado   | Fecha de creación                                                               |
| Acciones | Ver, Editar, Eliminar                                                           |
