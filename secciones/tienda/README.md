# Tienda

| Seccion                                         | Descripcion                                                 |

| [Menu](menu/)                                   | <sub>Administración del menú de navegación del header</sub> |
| [Administrador de Contenidos](content-manager/) | Editor visual de la pagina de inicio                        |
| [Contenidos Estaticos](contenidos-estaticos/)   | Páginas estáticas (FAQ, políticas, etc)                     |
| [Landings](landings/)                           | Landing pages personalizadas con editor de bloques          |
