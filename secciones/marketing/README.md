# Marketing

Descripción

Este módulo de Hermés sirve para:

1. [Sincronización de catálogo:](xml-feeds/) sirve para visualizar y crear la integración de catálogo para Google y Facebook.
2. [Atención al cliente:](atencion-al-cliente/) sirve para recibir mail de los clientes que envían desde la sección ¨contactos¨ y cada sitio podrá realizar la acción que desee con los mails recibidos.
3. [Suscriptos:](suscriptos.md) sirve para visualizar todos los suscriptos al sitio web, tanto de Newsletter como desde el footer.
