## Visualización en el front del sitio

Los usuarios pueden enviar sus consultas vía mail desde el apartado de contacto -> Escribinos

Al seleccionar "Escribinos", el cliente visualizará la pantalla de "Contacto" según el cliente.

Según el asunto que elija el cliente, será el formulario que se despliegue. Dentro de las opciones tenemos:

* Atención al cliente
* Venta mayorista
* RRHH&#x20;
* Prensa
* Seguimiento de envio
