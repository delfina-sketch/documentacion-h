# WhatsApp

#### Descripción

Esta herramienta permite informar al usuario, a través de WhatsApp, el estado de su pedido —ya sea para envío o retiro (pick up)— de forma clara y en tiempo real.

#### Beneficios <a href="#beneficios" id="beneficios"></a>

* **Mejora la experiencia del cliente**: El usuario recibe notificaciones en cada etapa del proceso (pago, confirmación, envío, entrega), lo que genera mayor confianza y mejora la percepción del servicio.
* **Optimiza tiempos y recursos**: Al brindar información proactivamente, disminuyen las consultas repetitivas, permitiendo que el equipo de Atención al cliente se enfoque en casos más específicos y personalizados.
* **Fomenta la fidelización**: Una atención ágil, transparente y cercana fortalece el vínculo con el cliente y aumenta las chances de recompra.

{% hint style="info" %}
Para utilizar la herramienta se deberá abonar un costo por mensaje. Para solicitar más detalles, comunicate con tu CX.
{% endhint %}

#### Implementación <a href="#implementacion" id="implementacion"></a>

Para poder activar las notificaciones por WhatsApp, es necesario seguir estos pasos previos:

1. **Consultar con el equipo de CX** sobre los costos asociados al servicio.
2. **Completar el** [**formulario**](https://form.asana.com/?k=v_sPm4Y17NxYHDezOEKt8A\&d=66032875349265) **de solicitud de set-up**, utilizando como asunto: **"Notificaciones por WhatsApp"**. En el formulario deberás incluir:
   * **Acceso al portfolio comercial de Meta**: Este paso es clave, ya que desde allí configuraremos el nombre del remitente para los mensajes. Para gestionar usuarios, ingresá en: [https://business.facebook.com](https://business.facebook.com/) > _Configuración_ > _Personas_.
   * **Nombre del remitente y logo/imagen** que se utilizará en las notificaciones.

Una vez realizada la configuración inicial, habilitaremos el módulo en Hermes para que puedas personalizar y activar los mensajes según tus necesidades.

#### Acceso <a href="#acceso" id="acceso"></a>

Desde el backoffice de Hermes, ir a Configuración > **Whatsapp**<br>
