# Configuracion

## Descripción

Este módulo tiene como objetivo poder configurar de forma simple y ordenada los diferentes modulos autoadministrables desde el backoffice de Hermés.

* Ajustes generales del sistema: datos de la marca, automatizaciones, logística, usuarios, objetivos de venta y facturación.

## Secciones

| Seccion                                 | Descripcion                                                      |
| --------------------------------------- | ---------------------------------------------------------------- |
| [General](general/)                     | Marca, contacto, SEO, redes sociales y configuracion del carrito |
| [Automatizaciones](automatizaciones.md) | Reportes automaticos y alertas de stock                          |
| [Logistica](logistica/)                 | Operadores logisticos, costos de envio y devoluciones            |
| [Usuarios / Administradores](usuarios/) | Gestion de usuarios del panel con roles y permisos               |
| [Objetivos](objetivos.md)               | Objetivos mensuales de venta para el dashboard                   |
| [Facturacion](facturacion/)             | Configuracion de facturacion digital                             |
