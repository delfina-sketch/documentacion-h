# Objetivos

## Acceso

**URL:** `/admin/configuracion/objetivos`

## Descripción

Esta sección permite ingresar los objetivos de venta para cada mes.

La información proporcionada se utiliza para generar el [reporte de ventas diario](automatizaciones.md), que se envía a todos los contactos configurados.

Así, al inicio de cada día, podremos contar con un reporte actualizado que nos brindará una visión rápida del estado de las ventas.
