# Creación de usuarios y asignación de permisos

## Descripción

Los administradores son las personas que tienen acceso a tu backoffice. Cada administrador tiene un permiso definido.&#x20;

{% hint style="info" %}
Es importante destacar que únicamente el usuario administrador posee la facultad de agregar nuevos usuarios y de asignarles diversos niveles de permisos.
{% endhint %}

## Creación de administradores

1\. Ingresar a Hermes -> Configuración -> Usuarios

2\. Completar el formulario -> Crear administrador.
