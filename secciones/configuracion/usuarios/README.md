# Usuarios

## Acceso:

**URL:** `/admin/administrators`

Gestión de usuarios administrativos del sistema con roles y permisos.

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-12 a la(s) 7.31.38 p. m..png" alt=""><figcaption></figcaption></figure>

## Tarjetas de resumen

| Metrica            | Color | Descripcion                 |
| ------------------ | ----- | --------------------------- |
| **Total usuarios** | Azul  | Total de usuarios del panel |
| **Admins**         | Azul  | Usuarios con rol Admin      |
| **Operadores**     | Verde | Usuarios con rol Operador   |

## Descripción

En este módulo, podrás crear, editar y eliminar los usuarios, así como configurar los permisos individuales que definirán su acceso a la tienda.

## Acciones

* **Exportar:** Descarga del listado
* [**+ Crear:**](https://app.gitbook.com/o/YqXgoQkfPbRYP7oDCRpr/s/4ixUGi3IayVNsRKEYT0I/~/edit/~/changes/38/secciones/configuracion/usuarios/creacion-de-usuarios-y-asignacion-de-permisos) Nuevo usuario administrador

## Columnas de la tabla

| Columna  | Ordenable | Descripcion                                         |
| -------- | --------- | --------------------------------------------------- |
| ID       | Si (desc) | Numero identificador (#)                            |
| Usuario  | Si        | Avatar (iniciales con color) + Nombre completo      |
| Email    | Si        | Direccion de email (con icono)                      |
| Rol      | No        | Badge: **Admin** (azul), **Operador**, etc.         |
| Tienda   | No        | Tienda asignada (si aplica)                         |
| Creado   | No        | Fecha de creacion                                   |
| Acciones | No        | Ver (ojo), Editar (lapiz), Eliminar (papelera roja) |
