# Facturacion

## Acceso

**URL:** `/admin/configuracion/facturacion`

Configuración del sistema de facturación digital.

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-13 a la(s) 10.04.12 a. m..png" alt=""><figcaption></figcaption></figure>

## **Descripción**

Este módulo nos permite configurar como realizaremos la facturación de cada orden que ingresa a nuestro sitio web. Puede ser de forma:

* Manual: donde cada marca se hace cargo de realizar la facturación de todas sus órdenes y en Hermés se facturan tocando ¨Generar Factura¨ en cada orden
* Automática: donde cada marca tiene que integrarse a una plataforma de facturación para que se conecte con Hermés y genere la factura.

## Configuración

Dentro del módulo se visualizará el detalle de la facturación configurada. Para modificar esta configuración, se tendrá que:

1. Ingresar en la sección de "Facturación".
2. Dentro de esta sección se visualizarán dos check de habilitado que corresponden a facturación automática y Factura A.

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-13 a la(s) 10.05.56 a. m..png" alt=""><figcaption></figcaption></figure>

El check de facturación digital, si esta desactivado la facturación se efectuará de manera manual. Caso contrario, activado (SI) la facturación se efectuará mediante la plataforma que se tenga integrada.

Para el caso de Factura B, consultar el [link](https://app.gitbook.com/o/YqXgoQkfPbRYP7oDCRpr/s/4ixUGi3IayVNsRKEYT0I/~/edit/~/changes/40/secciones/configuracion/facturacion/factura-b)

Para el caso de Factura A, consultar el [link](https://app.gitbook.com/o/YqXgoQkfPbRYP7oDCRpr/s/4ixUGi3IayVNsRKEYT0I/~/edit/~/changes/40/secciones/configuracion/facturacion/factura-a)
{% endhint %}

3. Luego de activar los checks que se deseen, deberán actualizar la configuración haciendo clic en ¨Guardar¨.&#x20;
