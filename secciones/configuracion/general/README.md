# General

## Acceso:

**URL:** `/admin/configuracion/general`

## Descripción

Desde este sub-módulo, vas a poder realizar las diferentes configuraciones para sitio. Dependiendo de la configuración a realizar, se podrá configurar:

* [Marca y contacto](marca-y-contacto.md): Datos básicos de la marca
* [Carrito](carrito.md): Configuración del carrito de compras
* [Banners Carrito](banners-carrito.md): Banners promocionales en el carrito
* [Pedido para regalo](pedido-para-regalo.md): Opción de pedido como regalo

## Acciones

* Botón **Guardar** para aplicar los cambios.
