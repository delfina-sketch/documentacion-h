# Banners carrito

## Descripción

Los clientes de Hermés podrán activar/desactivar un Banner en el carrito y carrito desplegable, en el que podrán comunicar promociones o beneficios para el usuario.&#x20;

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2024-08-28 a la(s) 20.45.38.png" alt="" width="563"><figcaption></figcaption></figure>

{% hint style="info" %}
**Las medidas** serían para donde dice "desktop" 2280 × 200 px. Donde dice "mobile" 670x200 px.

\
Se deben considerar:

* Banner desktop → Se utilizará para carrito desktop
* Banner mobile → Se utilizará para carrito desplegable desktop, carrito y carrito desplegable mobile.
{% endhint %}

## Funcionamiento

El banner se visualizará siempre que lo tengan con las imágenes cargadas, tanto en desktop como en mobile, la fecha configurada y los días activos:

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-12 a la(s) 5.11.31 p. m..png" alt=""><figcaption></figcaption></figure>

## Configuración

1. Ir al bloque ¨**Banners carrito"**
2. Subir la imagen tanto para desktop y para mobile
3. Configurar la fecha correspondiente.
4. Colocar los días que activos.
5. Seleccionar "Guardar".

El banner deja de visualizarse cuando no se cumple con la fecha configurada.

{% hint style="info" %}
IMPORTANTE: el banner de carrito y carrito desplegable, estará sujeto al diseño aprobado por cada marca, en caso de no tenerlo maquetado se deberá bajar el [formulario](https://form.asana.com/?k=v_sPm4Y17NxYHDezOEKt8A\&d=66032875349265) correspondiente al equipo de producto.
{% endhint %}
