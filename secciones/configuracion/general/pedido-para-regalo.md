# Pedido para regalo

## Descripción&#x20;

La funcionalidad de pedido para regalo le da al usuario la posibilidad de seleccionar “Agregar una bolsa de regalo” dentro del check out cuando solicite la misma dentro de su pedido.

## Configuración

1. Debemos ir al bloque “Pedido para regalo”
2. Habilitar o deshabilitar el check según corresponda a lo que se quiera configurar (prender o apagar la opción).
3. Seleccionar "Guardar” para guardar la configuración.
