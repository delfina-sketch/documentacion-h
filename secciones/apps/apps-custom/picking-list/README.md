# Picking List

Esta pantalla muestra la ficha de la aplicación "Picking List", que es la herramienta que usa el depósito y locales para organizar la preparación de los pedidos. Al instalar esta aplicación, se habilita la posibilidad de configuración de las pantallas de despacho tanto para el depósito central como para los locales.

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-05-28 a la(s) 16.35.46.png" alt=""><figcaption></figcaption></figure>

## Modo de uso

1. **Instalación de la app:** Se activa la herramienta en la plataforma.
2. **Creación de accesos:** Desde el módulo de administración, se crea el usuario con el rol de "Tienda" y se lo asigna al local que corresponda.

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-05-28 a la(s) 16.34.39.png" alt=""><figcaption></figcaption></figure>

3. **Configuración del local:** Se personaliza la pantalla de cada local ingresando a la sección de _Locales > Picking List_.

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-05-28 a la(s) 16.35.02.png" alt=""><figcaption></figcaption></figure>

Actualmente, como se ve en la imagen, la aplicación ya está "Configurada" y "Activa", lo que significa que las funciones para escanear productos, hacer facturas o remitos, imprimir etiquetas y despachar ya están listas para usar.
