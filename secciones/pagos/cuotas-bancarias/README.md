# Cuotas Bancarias

**URL:** `/admin/promotions-bancarias`

Este módulo permite configurar y activar o desactivar promociones bancarias, crear bancos y tarjetas. &#x20;

<figure><img src="../../../.gitbook/assets/image (11).png" alt=""><figcaption></figcaption></figure>

## Tarjetas de resumen

Al ingresar al módulo se mostrarán tarjetas con información mas relevante:

<table><thead><tr><th width="213.984375">Metrica</th><th>Descripcion</th></tr></thead><tbody><tr><td><strong>Total promociones</strong></td><td>Total de promociones creadas</td></tr><tr><td><strong>Activas</strong></td><td>Promociones vigentes</td></tr><tr><td><strong>Pendientes</strong></td><td>Promociones programadas</td></tr><tr><td><strong>Inactivas</strong></td><td>Promociones vencidas o desactivadas</td></tr></tbody></table>

También se podrá:

* Descargar el listado de promociones
* Crear nueva promociones bancarias
* Buscar promociones por nombre

También mostrará un listado con las promociones creadas.&#x20;
