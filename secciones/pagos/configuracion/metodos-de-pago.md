### Activación de medios de pago

{% hint style="info" %}
Antes de activar un medio de pago, asegurate que ya está integrado y configurado en tu tienda. Caso contrario, solicitá la integración mediante formulario.
{% endhint %}

Para activar un medio de pago, simplemente se activa el toggle del mismo:&#x20;

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-05 a las 15.19.15.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Para los medios de pago Transferencia y Rapipago / Pago fácil, se debe hacer una configuración adicional desde el App Store:&#x20;

*   Transferencia: Configurar los datos de la cuenta que se utilizarán para recepción de los pagos.

    * CBU
    * Alias
    * Banco
    * CUIT
    * Número de cuenta
    * Tipo de cuenta
    * Descuento (si se quiere ofrecer descuento por elegir el medio de pago): solo números


* Rapipago y Pago facil: Configurar los días para cancelación de orden.&#x20;



Para más información consultá App Store
{% endhint %}

### Carga de banners

Fiserv permite configurar un banner que se visualizará en el checkout.&#x20;

1. Clickear en el ícono de banner:&#x20;

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-05 a las 15.33.48.png" alt=""><figcaption></figcaption></figure>

2. Cargar el banner desktop y mobile y configurar:

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-05 a las 15.36.53.png" alt=""><figcaption></figcaption></figure>

* Prioridad: Es de uso interno. Se utiliza para ver los banners ordenados
* Fecha desde - hasta: fecha de activación y fin de la visualización del banner
* Días activos: Si se debe mostrar un día específico, marcarlo, sino marcar Todos.&#x20;
* Recurrencia: Si se activa el banner se mostrará los días configurados.&#x20;

3. Guardar los cambios. Se mostrará el banner cargado recientemente el cuál podrá ser activado o desactivado según preferencia. Se pueden configurar tantos banners como sean necesarios, pero solamente debe haber uno activo.
