# Configuración

**URL:** `/admin/configuracion-pagos`

En esta sección se podrán realizar 3 acciones:&#x20;

1. Activar y desactivar métodos de pago
2. Activar y desactivar el calculador de cuotas de la ficha
3. Ordenar banners de pago en el checkout

## Metodos de Pago disponibles

Lista draggable de metodos de pago con toggle de Activo/Inactivo:

| Metodo                       | Descripcion                       |
| ---------------------------- | --------------------------------- |
| MercadoPago Dinero en Cuenta | Pago con saldo de MercadoPago     |
| Gocuotas                     | Plataforma de cuotas              |
| Transferencia Bancaria       | Transferencia directa             |
| Talo                         | Pasarela de pagos                 |
| Modo                         | Billetera digital                 |
| Gift Card                    | Tarjetas de regalo                |
| MercadoPago                  | Gateway de MercadoPago            |
| Rapipago                     | Pago en efectivo                  |
| Pago Facil                   | Pago en efectivo                  |
| Payway                       | Pasarela de tarjetas (ex Decidir) |
| Oh Card                      | Tarjeta propia                    |
| Crypto                       | Pago con criptomonedas            |

> **Nota:** "Arrastra para reordenar. Los metodos sin instalar o configurar aparecen al final."

## Calculador de cuotas (panel derecho, collapsible)

| Campo     | Descripcion                            |
| --------- | -------------------------------------- |
| Cuotas    | Numero de cuotas (3, 6, 9, 12, 18, 24) |
| Desde ($) | Monto minimo para habilitar esa cuota  |
| Hasta ($) | Monto maximo                           |
| Activo    | Toggle Si/No (badge verde)             |

* Boton **+ Agregar fila** para mas configuraciones.

## Acciones

* Boton **Guardar** para aplicar los cambios.
