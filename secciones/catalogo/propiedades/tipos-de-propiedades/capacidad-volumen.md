# Capacidad (volumen)

Esta propiedad se suele utilizar en las fragancias o difusores de ambientes.

### Creación y edición de propiedad "Capacidad"

Para crear la propiedad Capacidad:

1. Ingresar a "Nueva propiedad"

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-03-02 a las 09.25.16.png" alt=""><figcaption></figcaption></figure>

2. Completar los campos:

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-03-02 a las 09.25.42.png" alt=""><figcaption></figcaption></figure>

* Nombre interno: Nombre interno para organización
* Nombre visible: Nombre que se verá en los filtros de la sábana
* Placeholder: Colocar el mismo nombre que el nombre visible
* Label: Colocar el mismo nombre que el nombre visible
* Categoría: Seleccionar si la propiedad es de variante o de ítem.
* Check "Filtrable en el sitio": Si se activa, el filtro figurará en la sábana de productos.

3. Una vez creada la propiedad, se deben crear los atributos de la misma desde "Agregar opción". Se podrán asignar una posición que reflejará el orden en la ficha de producto, y ordenar y borrar los atributos creados según preferencia:

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-03-02 a las 09.44.03.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Los atributos de una propiedad deben ser únicos y no deben repetirse.
{% endhint %}
