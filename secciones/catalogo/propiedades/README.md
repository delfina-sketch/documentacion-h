# Propiedades

**URL:** `/admin/propiedades`

Una propiedad es el/los atributo/s de un ítem (por ejemplo: color, talle).&#x20;

Desde este módulo podrás crear propiedades que servirán para:&#x20;

* la creación de prototipos (conjunto de ítems con los mismos atributos)
* la creación de ítems
* filtros de la sábana de ítems

### Acceso

Para acceder, ingresar a Catálogo > Propiedades:

## Tipos de propiedades

* **Choice:** Selector con opciones predefinidas (ej: talles S, M, L, XL)
* Las propiedades de tipo **Variante** generan combinaciones de SKUs
* Las propiedades de tipo **Producto** son informativas y no generan variantes
