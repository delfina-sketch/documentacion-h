# Catálogo

Sección para gestionar el catálogo completo de productos de la tienda. Incluye la gestión de productos, prototipos (plantillas), propiedades (atributos), importación masiva y configuración visual del catálogo.

<figure><img src="../../.gitbook/assets/image (31).png" alt=""><figcaption></figcaption></figure>

## Sub-secciones

| Seccion                         | Descripcion                                                           |
| ------------------------------- | --------------------------------------------------------------------- |
| [Productos](productos/)         | Listado y gestión de todos los productos                              |
| [Prototipos](prototipos/)       | Plantillas base que definen la estructura de productos                |
| [Propiedades](propiedades/)     | Atributos de productos y variantes (talle, color, etc.)               |
| [Carga Masiva](carga-masiva/)   | Importación de productos via Excel y de imágenes via Dropbox          |
| [Configuración](configuracion/) | Opciones visuales: out of stock, cucardas, solapas descriptivas, etc. |
