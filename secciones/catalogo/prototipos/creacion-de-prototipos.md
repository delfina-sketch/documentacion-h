# Creación de prototipos

Para crear prototipos es necesario que previamente se hayan creado las propiedades del producto (por ejemplo: talle y color).

1. Ingresar a Catálogo > Prototipos > Nuevo prototipo:&#x20;

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-02 a las 09.07.34.png" alt="" width="563"><figcaption></figcaption></figure>

2. Completar los campos:

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-02 a las 09.08.39.png" alt="" width="563"><figcaption></figcaption></figure>

* Nombre: Nombre interno para organización
* Display: Nombre que se visualiza al crear un ítem.&#x20;
* Propiedades de ítem: Seleccionar la propiedad de ítem que le corresponda. Ej: Tipo
* Propiedades de variante: Seleccionar la propiedad de variante que le corresponda. Ej: Color

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-02 a las 09.11.28.png" alt=""><figcaption></figcaption></figure>

3. Una vez completado los campos, guardar haciendo click en “Crear Prototipo”

{% hint style="info" %}
Consideraciones adicionales:

* Un prototipo debe tener como mínimo una propiedad creada.&#x20;
* Una vez creado el prototipo y asignado a los productos en la matriz de se deben eliminar ya que “rompería” la ficha de los productos. Tampoco debe modificarse.
{% endhint %}
