# Carga Masiva

**URL:** `/admin/products/import`

Este módulo permite crear ítems de manera masiva mediante archivo Excel, y también cargar imágenes de ítems mediante la integración con Dropbox.&#x20;

## Acceso

Para acceder, ingresar a Catálogo > Carga masiva:

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-03-02 a las 09.55.07.png" alt=""><figcaption></figcaption></figure>

Al ingresar al módulo, la carga masiva y la carga de fotos estará dividida en dos pestañas, siendo el orden correcto a seguir:&#x20;

1. La carga masiva de productos
2. La carga de fotos

En los siguientes módulos se explicará de forma detallada el paso a paso para realizar ambas gestiones.
