# Importación de imágenes

La pestaña "Imagenes desde Dropbox" permite sincronizar imágenes de productos desde una carpeta compartida de Dropbox, asociandolas automáticamente a cada SKU.

{% hint style="info" %}
Requisitos previos:

* **Matriz creada y cargada previamente:** Los productos deben haber sido creados en Hermes. 👉 Más info: Carga masiva de productos
* **Cuenta de Dropbox integrada a Hermes:** Es obligatorio contar con una cuenta de Dropbox y haberla integrado correctamente a Hermes. Más info: DropBox
* **Naming de fotos:** Las fotos deben tener un nombre específico con la siguiente estructura: **Raíz foto + guión + posición.** Ejemplo: Si la raíz foto del producto es `SS20VES0126`, los archivos deben llamarse: `SS20VES0126-1`, `SS20VES0126-2`, etc.
* **Proporción / Tamaño:** Las imágenes deben respetar la proporción establecida para la web (ej.: **2:3**)
{% endhint %}

El proceso tiene dos pasos:&#x20;

1. Organización las carpetas en Drobbox.
2. Carga de imágenes.
