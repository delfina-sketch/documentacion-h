# Carga masiva de productos

Para realizar la carga masiva de productos se deberán seguir los siguientes pasos:&#x20;

1. Armado de matriz, que incluye:&#x20;
   1. Descargar la tabla de equivalencias
   2. Armar archivo excel con la información de los productos
2. Subir la matriz a Hermes
3. Mapear columnas: Asociar las columnas del Excel con los campos del sistema
4. Vista previa: Revisar los datos antes de importar
5. Confirmar: Ejecutar la importación
