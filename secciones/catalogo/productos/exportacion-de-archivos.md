# Exportación de archivos

Esta herramienta sirve para descargar archivos de información de ítems. Es posible descargar todo el catálogo, o seleccionar algunos productos mediante el check de la izquierda de cada uno.

#### Exportación de productos especificos

1. Tildar los ítems de los que se quiere importar información con el check de la izquierda:

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 14.59.01.png" alt="" width="563"><figcaption></figcaption></figure>

2. Click en el botón "Exportar"

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 16.17.52.png" alt="" width="563"><figcaption></figcaption></figure>

2. Confirmar la acción con el botón "Solicitar exportación". Se enviará un email con el archivo adjunto al correo utilizado para log in.&#x20;

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 14.37.16.png" alt="" width="563"><figcaption></figcaption></figure>

#### Exportación de todos los productos del catálogo

1. Para exportar todos los productos, directamente hacer click en "Exportar"

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 16.17.52 (1).png" alt="" width="563"><figcaption></figcaption></figure>

2. Confirmar la acción con el botón "Solicitar exportación". Se enviará un email con el archivo adjunto al correo utilizado para log in.&#x20;

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 14.37.16.png" alt="" width="563"><figcaption></figcaption></figure>
