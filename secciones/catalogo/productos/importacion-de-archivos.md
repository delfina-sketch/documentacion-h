# Importación de archivos

Esta herramienta sirve para subir archivos de actualización de productos (precios, stock, cucardas, etc).&#x20;

{% hint style="info" %}
En todos los casos el archivo debe tener formato .csv
{% endhint %}

1. Click en el botón "Importar"
2. Elegir una opción según corresponda:
   * Actualizar variantes: si el archivo es de variantes
   * Actualizar ítems: si el archivo es de ítems

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 14.40.18.png" alt=""><figcaption></figcaption></figure>

3. Se solicitará que elija una acción y se adjunte el archivo:&#x20;

* Pisar: reemplaza la información actual por la nueva del csv
* Sumar: adiciona información a la ya existente. Sólo aplica a archivos de stock o categorías.
* Restar: sustrae información de la ya existente. Sólo aplica a archivos de stock o categorías.

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 14.43.43.png" alt=""><figcaption></figcaption></figure>

4. Adjuntar el archivo.&#x20;
5. Guardar y procesar.&#x20;

De estar todo correcto, el archivo comenzará a procesarse. Caso contrario, se informará "CSV inválido" y deberá revisarse su armado.&#x20;
