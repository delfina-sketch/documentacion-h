# SEO (Optimización para buscadores)

La pestaña **SEO** permite optimizar cómo se verá el producto en Google y mejorar su posicionamiento en buscadores.

Incluye una **vista previa en Google** que muestra cómo aparecerá el producto en los resultados de búsqueda. La configuración es opcional pero se recomienda completar.

1. Vista previa en Google: Se muestra una preview de los resultados de búsqueda de Google:

* Título del producto (clickeable)
* URL del producto
* Descripción que aparecerá en el resultado

Esta vista se actualiza automáticamente a medida que se completan los campos SEO.

***

2. Meta título (opcional): Es el título que se mostrará en Google. Puede ser diferente al título interno del producto. Recomendado: **50 a 60 caracteres**.

{% hint style="info" %}
Buenas prácticas:

* Incluir nombre del producto.
* Agregar marca si corresponde.
* Incluir palabra clave principal.
* Evitar títulos demasiado largos (Google los corta)
  * **Ejemplo:** Camisa de Lino Blanca Mujer | Marca X&#x20;
{% endhint %}

3\. Meta Descripción (opcional): Es el texto descriptivo que aparece debajo del título en Google.

* Recomendado: **150 a 160 caracteres**.

{% hint style="info" %}
Buenas prácticas:

* Describir el producto de forma clara.
* Incluir beneficios.
* Usar lenguaje persuasivo.
* Incluir palabras clave relevantes.
  * **Ejemplo:** Camisa de lino blanca para mujer, liviana y fresca. Ideal para verano. Envíos a todo el país y hasta 6 cuotas sin interés.
{% endhint %}

4\. Meta Keywords (opcional): Permite ingresar palabras clave separadas por comas. Actualmente Google casi no las utiliza para posicionamiento, pero pueden servir para otros buscadores.

Formato: camisa lino blanca, camisa mujer verano, camisa fresca mujer

{% hint style="info" %}
Recomendaciones SEO importantes

* No duplicar meta títulos entre productos.
* No copiar y pegar la misma meta descripción en todos los productos.
* Usar palabras que los clientes realmente buscarían.
* Evitar mayúsculas innecesarias.
* No abusar de palabras clave repetidas.
{% endhint %}
