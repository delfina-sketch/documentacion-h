# Información básica

En esta pestaña se configura la información general del producto:

<figure><img src="../../../../.gitbook/assets/image (16).png" alt=""><figcaption></figcaption></figure>



1\. Prototipo (obligatorio): Seleccionar el **prototipo correspondiente** desde el desplegable.

{% hint style="info" %}
El prototipo define las propiedades disponibles del producto (ej: color, talle, etc.).
{% endhint %}

2. Título (obligatorio): Ingresar el nombre del producto. Ejemplo: Camisa Lino Blanca
3. Slug (obligatorio) : Se genera automáticamente a partir del título. Se puede editar manualmente si se desea optimizar la URL. Ejemplo: el slug de **https://template.pow.la/productos/remera-negra** es remera-negra, podría ser también remera\_negra-123

{% hint style="info" %}
El slug no permite caracteres especiales, excepto guión medio (-) y guión bajo (\_)
{% endhint %}

4. Categorías (obligatorio):
   1. Hacer clic en el buscador.
   2. Escribir al menos 2 caracteres.
   3. Seleccionar la categoría correspondiente.

{% hint style="info" %}
Un producto puede tener **una o más categorías**.
{% endhint %}

5. Descripción breve (opcional): Agregar un resumen corto del producto.

{% hint style="info" %}
Esta descripción no es visible en el front, pero puede servir para agregar palabras clave que optimiza las búsqueda de productos.
{% endhint %}

6. Descripción completa (obligatorio): Agregar el detalle completo del producto:
   * Características
   * Materiales
   * Beneficios
   * Información adicional
7. Imagen principal del producto (obligatorio):  Esta será la imagen que se muestra en los listados de productos.
   1. Hacer clic en **“Seleccionar archivo”**.
   2. Elegir la imagen desde la computadora.
   3. Confirmar la carga.
8. Readysize (opcional): Si el sitio tiene integración con ReadySize:
   * **Código producto**: ingresar el código correspondiente.
   * **Fit Type**: indicar el tipo de ajuste.
9.  Pestañas descriptivas (opcional): Permite agregar información adicional en formato de pestañas dentro de la ficha del producto.

    Ejemplos de pestañas recomendadas:

    * Composición
    * Cuidados
    * Envíos y cambios
    * Medios de pago

    Para agregar una:

    1. Hacer clic en **“Agregar primera pestaña”**
    2. Completar título y contenido.
    3. Guardar.
