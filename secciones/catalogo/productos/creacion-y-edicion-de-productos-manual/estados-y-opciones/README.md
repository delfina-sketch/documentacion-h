# Estados y opciones

En esta pestaña se configura la visibilidad del producto, su comportamiento en tienda y las etiquetas promocionales.

<figure><img src="../../../../../.gitbook/assets/image (41).png" alt=""><figcaption></figcaption></figure>

1. Estado del producto (obligatorio): Activa o desactiva la visibilidad del producto en la tienda.

* Si está **activado**, el producto se muestra.
* Si está **desactivado**, el producto no será visible para los clientes.

2. Outlet (opcional): Marcar si el producto pertenece a la sección de outlet. Caso contrario se asume que es de main store.
3. Out of stock (opcional):
   1. Muestra el producto en sábana y en ficha como sin stock.&#x20;
   2. Activa la funcionalidad que permite que el usuario deje su email y sea notificado sobre el reingreso del producto.&#x20;

{% hint style="info" %}
Para más información consultá "Out of stock"
{% endhint %}

4. Buscador activo (opcional): Define si el producto aparece en los resultados de búsqueda del buscador. Si se desactiva, el producto no aparecerá cuando el usuario lo busque.
5. Excluír promociones bancarias (opcional): Determina si una promoción bancaria será visible en el checkout según los productos del carrito. Para que funcione correctamente, también debe activarse la opción "Solo nacionales" al crear o editar una promoción bancaria. Si al menos uno de los productos en el carrito tiene este check activo, la promoción no se mostrará disponible para elegir.
6. Cucardas (opcional): Permiten destacar el producto con badges o etiquetas en la ficha y en la sábana.
   1. Oferta: Muestra cucarda de oferta.
   2. Descuento: Muestra etiqueta de **descuento**. El porcentaje se obtiene por la diferencia entre el precio full y el precio tachado.
   3. Nuevo: Muestra una cucarda de texto "NUEVO"
   4. Últimas unidades: Muestra etiqueta de **Últimas unidades o Last units**, correspondiente a la funcionalidad de Últimas unidades.

{% hint style="info" %}
Para más información sobre la funcionalidad, consultá "Últimas unidades"
{% endhint %}

e. Envío gratis: Muestra cucarda de **envío gratis**.

f. Exclusivo: Indica que el producto es exclusivo online.

g. Pre-order: Muestra una cucarda de pre-order.&#x20;

{% hint style="info" %}
Para más información sobre la funcionalidad, consultá "Pre order"
{% endhint %}

7. Cucardas personalizadas (opcional): El sistema permite agregar cucardas de texto y de imagen que pueden utilizarse para campañas específicas (Ej: Hot Sale, Cybermonday, Sale, etc.).
   1. Cucarda de texto #1: es una cucarda de texto dinámica. Muestra el contenido configurado en formato de texto.
   2. Cucarda de texto #2: es una segunda cucarda de texto dinámica. Muestra el contenido configurado en formato de texto.
   3. Cucarda imagen #1: es una cucarda de imagen. El formato de la imagen debe ser PNG y no debe superar la medida de 50x50px.
   4. Cucarda imagen #2: es una segunda cucarda de imagen. El formato de la imagen debe ser PNG y no debe superar la medida de 50x50px.

{% hint style="info" %}
## La visualización de cucardas se encuentra sujeta a maquetado.&#x20;
{% endhint %}
