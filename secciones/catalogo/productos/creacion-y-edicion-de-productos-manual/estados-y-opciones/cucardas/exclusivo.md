# Exclusivo

La cucarda **Exclusivo online** se utiliza para destacar productos que únicamente se encuentran en la tienda online.

<figure><img src="../../../../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 11.37.13.png" alt="" width="344"><figcaption></figcaption></figure>

La misma se visualiza en:

* SKU carousel (home)
* Ficha
* Sábana
* Productos relacionados
* Favoritos

{% hint style="info" %}
La visualización de la cucarda se encuentra sujeta a maquetado. Para solicitarlo, envianos un formulario o contactate con tu CX asignado.
{% endhint %}

### Activación manual de cucarda Exclusivo

1. Ingresar al producto > Estados y opciones
2. Activar la cucarda mediante el toggle

<figure><img src="../../../../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 11.39.28.png" alt=""><figcaption></figcaption></figure>

3. Guardar los cambios.

{% hint style="info" %}
La cucarda también se puede activar masivamente. Consultá "Actualización de productos masiva (CSV)" para más información.&#x20;
{% endhint %}
