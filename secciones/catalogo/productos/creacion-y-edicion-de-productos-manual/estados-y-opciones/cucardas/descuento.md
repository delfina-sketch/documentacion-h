# Descuento

La **cucarda Descuento** muestra visualmente el porcentaje de rebaja del producto.\
El cálculo se realiza automáticamente tomando la diferencia entre el **precio full** y el **precio especial** (siempre que este último sea distinto de 0).

Por ejemplo:

La Remera Genérica cuenta con los siguientes precios:

* Full price: $1000
* Special price/precio tachado: $500

La cucarda que se visualizará dirá: **50% OFF**

<figure><img src="../../../../../../.gitbook/assets/image (21).png" alt="" width="344"><figcaption></figcaption></figure>

La misma se visualiza en:

* SKU carousel (home)
* Ficha
* Sábana
* Productos relacionados
* Favoritos

{% hint style="info" %}
La visualización de la cucarda se encuentra sujeta a maquetado. Para solicitarlo, envianos un formulario o contactate con tu CX asignado.
{% endhint %}

### Activación de la cucarda Descuento

1. Ingresar al producto > Estados y opciones
2. Activar la cucarda mediante el toggle

<figure><img src="../../../../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 11.48.45.png" alt=""><figcaption></figcaption></figure>

3. Guardar los cambios.&#x20;

{% hint style="info" %}
La cucarda también se puede activar masivamente. Consultá "Actualización de productos masiva (CSV)" para más información.&#x20;
{% endhint %}
