# Envío gratis

La cucarda de **Envío gratis** está vinculada a la configuración de envío gratis a partir de cierto monto. Permite activar de forma ágil una etiqueta informativa en los productos que cuentan con este beneficio.

Con esta configuración activa, el sistema mostrará automáticamente la cucarda en todos los productos a los que aplique el beneficio, según la configuración vigente de envíos.

La cucarda de Envío Gratis se visualiza en:

* Sábana
* Ficha&#x20;
* SKU carousel&#x20;
* Productos relacionados
* Favoritos&#x20;

{% hint style="info" %}
La visualización de la cucarda se encuentra sujeta a maquetado. Para solicitarlo, envianos un formulario o contactate con tu CX asignado.
{% endhint %}

### Activación de cucarda Envío gratis

1. Ingresar al producto > Estados y opciones
2. Activar el checkbox **“Envío gratis”**.

<figure><img src="../../../../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 13.21.55.png" alt=""><figcaption></figcaption></figure>

3. Hacer clic en **“Guardar”**.
