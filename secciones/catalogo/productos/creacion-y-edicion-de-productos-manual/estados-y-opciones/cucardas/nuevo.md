# Nuevo

La cucarda **NUEVO** se utiliza para diferenciar los últimos productos ingresados en la tienda.

<p align="center"><br><img src="../../../../../../.gitbook/assets/image (22).png" alt=""></p>

La misma se visualiza en:

* SKU carousel (home)
* Ficha
* Sábana
* Productos relacionados
* Favoritos

{% hint style="info" %}
La visualización de la cucarda se encuentra sujeta a maquetado. Para solicitarlo, envianos un formulario o contactate con tu CX asignado.
{% endhint %}

### Activación de la cucarda NUEVO

1. Ingresar al producto > Estados y opciones
2. Activar la cucarda mediante el toggle y configurar las fechas de inicio y fin de visualización de la cucarda:

<figure><img src="../../../../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 12.18.21 (1).png" alt=""><figcaption></figcaption></figure>

3. Guardar los cambios.&#x20;

{% hint style="info" %}
La cucarda también se puede activar masivamente. Consultá "Actualización de productos masiva (CSV)" para más información.&#x20;
{% endhint %}
