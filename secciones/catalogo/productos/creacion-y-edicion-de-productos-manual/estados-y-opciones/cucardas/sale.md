# Sale

La cucarda **SALE** se utiliza para diferenciar los productos con descuento.

<p align="center"><br><img src="../../../../../../.gitbook/assets/image (20).png" alt=""></p>

La misma se visualiza en:

* SKU carousel (home)
* Ficha
* Sábana
* Productos relacionados
* Favoritos

{% hint style="info" %}
La visualización de la cucarda se encuentra sujeta a maquetado. Para solicitarlo, envianos un formulario o contactate con tu CX asignado.
{% endhint %}



### Activación manual de cucarda SALE

1. Ingresar al producto > Estados y opciones
2. Activar la cucarda mediante el toggle

<figure><img src="../../../../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 11.34.33 (1).png" alt=""><figcaption></figcaption></figure>

3. Guardar los cambios

{% hint style="info" %}
La cucarda también se puede activar masivamente. Consultá "Actualización de productos masiva (CSV)" para más información.&#x20;
{% endhint %}
