# Cucarda de texto #1 y #2

Las **cucardas de texto** permite configurar textos personalizados. Recomendamos utilizarlas en los casos donde no haya una cucarda específica con la información que se necesita comunicar y que sea de texto corto.

La misma se visualiza en:

* SKU carousel (home)
* Ficha
* Sábana
* Productos relacionados
* Favoritos

<figure><img src="../../../../../../.gitbook/assets/image (18).png" alt="" width="339"><figcaption></figcaption></figure>

{% hint style="info" %}
La visualización de la cucarda se encuentra sujeta a maquetado. Para solicitarlo, envianos un formulario o contactate con tu CX asignado.
{% endhint %}

### &#x20;Activación manual de cucardas de texto

1. Ingresar al ítem > Estados y opciones
2. Activar la cucarda de texto mediante el toggle de "Habilitado"

<figure><img src="../../../../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 11.25.58.png" alt=""><figcaption></figcaption></figure>

3. Cargar el texto deseado
4. Guardar los cambios.

{% hint style="info" %}
La cucarda también se puede activar masivamente. Consultá "Actualización de productos masiva (CSV)" para más información.&#x20;
{% endhint %}
