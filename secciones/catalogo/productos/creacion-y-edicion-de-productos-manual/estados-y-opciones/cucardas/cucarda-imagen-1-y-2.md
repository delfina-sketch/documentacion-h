# Cucarda imagen #1 y #2

La cucarda imagen permite cargar una imagen en formato PNG. En general se suele utilizar para destacar productos en eventos como Hot Sale, Cybermonday, etc.

La misma se visualiza en:

* Ficha
* Sábana
* SKU carousel
* Productos relacionados
* Favoritos

<figure><img src="../../../../../../.gitbook/assets/image (17).png" alt="" width="341"><figcaption></figcaption></figure>



{% hint style="info" %}
La visualización de la cucarda se encuentra sujeta a maquetado. Para solicitarlo, envianos un formulario o contactate con tu CX asignado.
{% endhint %}

### Activación de cucarda PNG manual

Se debe definir si la visualización será general (en todos los ítems) o sólo en ítems específicos.&#x20;

#### **Activación de cucarda en todos los ítems**

1. Ingresar a Configuración > Cucardas PNG

<figure><img src="../../../../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 11.13.15.png" alt=""><figcaption></figcaption></figure>

2. Habilitar la cucarda y cargar las imágenes:

<figure><img src="../../../../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 11.16.36.png" alt=""><figcaption></figcaption></figure>

3. Guardar los cambios.

#### **Activación de cucarda en ítems específicos**

1. Ingresar a cada producto > pestaña "Estados y opciones"
2. Activar el check de "Cucarda Imagen #1" y "Cucarda Imagen #2" y cargar las imágenes

<figure><img src="../../../../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 11.20.42.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Recomenaciones adicionales

* La imagen debe tener formato PNG y no medir más de 50x50px.
{% endhint %}

3. Guardar los cambios.

{% hint style="info" %}
La cucarda también se puede activar masivamente. Consultá "Actualización de productos masiva (CSV)" para más información.&#x20;
{% endhint %}
