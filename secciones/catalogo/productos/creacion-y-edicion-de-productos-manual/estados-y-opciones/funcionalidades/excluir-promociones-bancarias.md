# Excluír promociones bancarias

La funcionalidad **Excluir promociones bancarias** permite definir qué productos no participarán de las promociones asociadas a medios de pago bancarios.

Al activarla, el producto quedará fuera de los beneficios como cuotas sin interés, descuentos con determinados bancos o tarjetas, aunque dichas promociones estén vigentes en la tienda.

Esto brinda mayor control sobre la rentabilidad y la estrategia comercial, permitiendo excluir artículos específicos —por ejemplo, productos en oferta, con márgenes reducidos o pertenecientes a categorías especiales— de este tipo de acciones promocionales.

### Activación

1. Ingresar al producto
2. Activar el toggle de "Excluír promociones bancarias"

<figure><img src="../../../../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 13.13.44.png" alt=""><figcaption></figcaption></figure>

3. Guardar los cambios.
4. Ingresar a las promociones bancarias de las que se desea excluir el producto y activar el toggle **“Solo nacionales”** dentro de la configuración correspondiente.

<figure><img src="../../../../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 13.16.23.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Es posible activar la funcionalidad en los productos masivamente. Consultá "Actualización de productos masiva (CSV)" para más información.&#x20;
