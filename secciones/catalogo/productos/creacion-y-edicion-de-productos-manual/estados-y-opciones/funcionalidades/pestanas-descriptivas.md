# Pestañas descriptivas

Las **pestañas descriptivas** permiten incorporar información adicional en la ficha de producto, organizándola de manera clara y accesible para el usuario.

A través de estas pestañas, es posible detallar contenido relevante como especificaciones técnicas, composición, instrucciones de uso y cuidado, características destacadas u otra información que facilite la decisión de compra.

<figure><img src="../../../../../../.gitbook/assets/image (27).png" alt="" width="552"><figcaption></figcaption></figure>

### Activación de las solapas

1.  **Manual:**&#x20;

    1. Ingresar al producto y luego a la pestaña "Información básica"
    2. Ir a "Agregar primera pestaña"
    3. Cargar el nombre de la pestaña y el contenido. Ambos elementos se visualizarán en la ficha. Se pueden crear tantas pestañas como se tengan maquetadas.

    <figure><img src="../../../../../../.gitbook/assets/descarga (1).png" alt="" width="563"><figcaption></figcaption></figure>
2. **Masiva**: Las solapas deben tener el mismo contenido.&#x20;

{% hint style="info" %}
Consultá la sección "[Configuración](../../../../configuracion/)" de catálogo para más información.&#x20;
{% endhint %}

3. **Por csv:** las solapas pueden tener diferente contenido.

{% hint style="info" %}
Consultá la sección "[Actualización de productos por CSV](https://app.gitbook.com/o/YqXgoQkfPbRYP7oDCRpr/s/4ixUGi3IayVNsRKEYT0I/~/edit/~/changes/4/secciones/catalogo/productos/actualizacion-de-productos-masiva-csv)" para más información.&#x20;
{% endhint %}
