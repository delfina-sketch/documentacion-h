# Buscador activo

La funcionalidad **Buscador Activo** permite configurar qué productos pueden mostrarse en los resultados del buscador de la tienda.

A través de esta opción, es posible definir y restringir la visibilidad de determinados productos dentro de las búsquedas, asegurando que solo aparezcan aquellos que se deseen destacar o habilitar para esta sección.

{% hint style="info" %}
**Consideraciones:**

* Todos los productos que tengan stock, foto y estén habilitados, así como los productos nuevos que se carguen, se mostrarán siempre con el estado ‘True’. Esto indica que estos productos son 'visibles' en el buscador.
* Antes de utilizar esta funcionalidad, es necesario solicitar al equipo de IT que cambie el estado de todos los productos a ‘False’. El estado ‘False’ desactiva el ítem en el buscador, aunque el producto seguirá visible en la sábana de productos.
* Cabe destacar que esta funcionalidad se aplica a nivel de ítem y no se puede utilizar para variantes específicas de un producto.
{% endhint %}

### **Activación manual**

1. Ingresar al producto > Estados y opciones.
2. Activar el toggle de "Buscador activo"

3. Guardar los cambios.

{% hint style="info" %}
Es posible activar la funcionalidad en los productos masivamente. Consultá "Actualización de productos masiva (CSV)" para más información.&#x20;
{% endhint %}
