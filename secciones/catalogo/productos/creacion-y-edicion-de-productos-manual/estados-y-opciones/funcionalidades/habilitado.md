# Habilitado

La opción **Habilitado** define si el ítem será visible en la tienda online.

Para que un ítem esté visible en la web, se deben cumplir los siguientes requisitos:&#x20;

* Tener fotos en la primer variante de cada color
* Tener stock
* Tener precio
* Estar habilitado
  * Si está **activado**, el producto se muestra y puede ser comprado (si tiene stock y precio cargado).
  * Si está **desactivado**, el producto no será visible para el usuario, aunque permanezca creado en el sistema.

### Activación manual

1. Ingresar al producto > Estados y opciones.
2. Activar el toggle de "Habilitado"

<figure><img src="../../../../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 12.40.14.png" alt=""><figcaption></figcaption></figure>

3. Guardar los cambios.
