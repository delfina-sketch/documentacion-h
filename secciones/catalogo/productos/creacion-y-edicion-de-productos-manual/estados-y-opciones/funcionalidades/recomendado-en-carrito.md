# Recomendado en carrito

Esta opción permite definir qué variantes se mostrarán como **ítems** **recomendados en el carrito**, con el objetivo de incentivar la venta complementaria.

#### Configuración <a href="#configuracion" id="configuracion"></a>

Existen dos formas de configurar los productos sugeridos del carrito:

#### Por variante

1. Ingresar al ítem > Variantes y buscar la variante correspondiente.
2. Activar el check **“Recomendado en carrito”** en la variante deseada.

<figure><img src="../../../../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 13.52.25.png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
También es posible configurar sugeridos a nivel categoría. Para ello, se debe seleccionar una categoría específica.

Para más información sobre esta configuración, consultar la sección [Categoría recomendada en carrito](https://app.gitbook.com/o/YqXgoQkfPbRYP7oDCRpr/s/4ixUGi3IayVNsRKEYT0I/~/edit/~/changes/4/secciones/configuracion/general/carrito/categoria-recomendada-en-carrito) de Configuración General.
