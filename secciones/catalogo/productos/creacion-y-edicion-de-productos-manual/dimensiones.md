# Dimensiones

La pestaña **Dimensiones** permite configurar el peso y tamaño del producto.

Esta información es fundamental y obligatoria para que el sistema calcule correctamente el costo de envío.

{% hint style="info" %}
**Todos los campos son obligatorios si el cálculo de envío está integrado con correo o logística automática.**
{% endhint %}

Las medidas deben cargarse con las siguientes unidades:

* **Peso (obligatorio):** en gramos (gr). No colocar "gr" ni puntos. Solo números.
* **Ancho (obligatorio):** en centímetros (cm)
* **Alto (obligatorio):** en centímetros (cm)
* **Profundidad (obligatorio):** en centímetros (cm)

{% hint style="info" %}
No colocar "gr" ni puntos. Solo números.
{% endhint %}

#### ¿Qué medidas cargar?

Se recomienda cargar las medidas del producto embalado, no solo del producto suelto.\
Es decir, considerar bolsa, caja o packaging final. Esto evita diferencias en el cálculo del envío.
