# Importar archivo CSV

Una vez que el CSV está listo para ser importado, se deberá importar desde Productos > Importar.&#x20;

{% hint style="info" %}
Para más detalles de como importar un archivo, consultá [Importación de archivos](https://app.gitbook.com/o/YqXgoQkfPbRYP7oDCRpr/s/4ixUGi3IayVNsRKEYT0I/~/edit/~/changes/4/secciones/catalogo/productos/importacion-de-archivos).
{% endhint %}

Si el archivo se procesó sin errores, se podrá ir viendo el porcentaje (%) de avance.&#x20;

En caso contrario, se informará la notificación "Error al procesar archivo"

<figure><img src="../../../../.gitbook/assets/Captura de pantalla 2026-02-27 a las 17.10.52.png" alt="" width="563"><figcaption></figcaption></figure>

Algunos de los errores más comunes pueden ser:

* CSV separado con "," (_comas_) en vez de ";" (_punto y comas_)
* Nombres de columnas con errores de ortografía (revisar [archivo de modificaciones vía csv ](https://docs.google.com/spreadsheets/d/1watxVk29qfmzZMhgnbFIXE5oP-Z3hQ1o9QIHvMPvmSI/edit?gid=0#gid=0))
* Celdas con caracteres no visibles (comilla simple por ejemplo)
* Columnas vacías pero con formato / fórmulas
