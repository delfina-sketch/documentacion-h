# Obtener CSV de base

El primer paso consiste en el armado del archivo. Por lo general, a la hora de realizar modificaciones sobre productos en la web, lo más conveniente será descargar un CSV de dichos productos, para posteriormente realizar las modificaciones, y volver a cargarlo.

{% hint style="info" %}
Consultá "Exportación de archivos" para más información sobre como exportar un csv.
{% endhint %}

Para poder armar un csv, es necesario diferenciar correctamente las características de un ítem y de una variante.&#x20;

* **Item:** refiere a un producto genérico, que no distingue entre variantes. Por ejemplo, "_**Camisa Ford**_" (sin especificar color o talle aún).
* **Variante:** son aquellas variaciones de producto que se distinguen según el valor de sus propiedades (Por ejemplo, "_**Camisa Ford Roja XL**_" es una variante de la Camisa Ford).
