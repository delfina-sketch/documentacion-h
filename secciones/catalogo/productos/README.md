# Productos

**URL:** `/admin/products`

A continuación se detalla el funcionamiento de la pantalla **Productos**, desde donde se gestiona el catálogo completo del ecommerce.

### 1. Resumen general (cards superiores)

En la parte superior se visualizan métricas generales del catálogo que permiten tener una visión rápida del estado del catálogo:

### 2. Acciones principales

Ubicadas en el margen superior derecho:

* **Importar**: permite cargar productos de manera masiva (ej. subir stock, habilitar productos, etc).
* **Exportar**: descarga el listado de productos en archivo para análisis o edición externa.
* **Nuevo producto**: crea un producto manualmente desde cero.

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-02-26 a las 17.55.15.png" alt=""><figcaption></figcaption></figure>

### 3. Buscador y filtros

Debajo de las métricas se encuentra:

* **Buscador por título o SKU**: permite localizar productos rápidamente.
* **Botón de filtros**: habilita filtros adicionales:&#x20;
  * Categorías
  * Estado (habilitado / deshabilitado)

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-02-26 a las 17.56.28.png" alt=""><figcaption></figcaption></figure>



### 4. Listado de productos

La grilla muestra la información principal de cada producto:

* **Producto**
  * Nombre del producto.
  * Descripción breve.
* **Categorías**
  * Categorías asignadas al producto.
  * Si hay más de las visibles, se indica con “+X”.
* **Stock**
  * Cantidad total disponible.
  * Total de stock disponible.
* **Precio**
  * Precio full.
* **Estado**
  * **Habilitado**: visible y disponible para la venta (si tiene stock).
  * **Deshabilitado**: no visible en el ecommerce.
* **Acciones**
  * Ver producto.
  * Editar producto.

<figure><img src="../../../.gitbook/assets/Captura de pantalla 2026-02-26 a las 17.56.12 (1).png" alt=""><figcaption></figcaption></figure>

### 5. Gestión de productos

Desde esta pantalla es posible:

* Habilitar o deshabilitar productos.
* Editar información (precio, stock, categorías, etc.).
* Revisar rápidamente el estado comercial del catálogo.
* Identificar productos sin stock.
* Ver productos con múltiples variantes.

***

### 6. Paginado

En el pie de la tabla se indica:

* Rango de productos visualizados (ej. “Mostrando 1 a 20 de 193”).
* Navegación por páginas.
