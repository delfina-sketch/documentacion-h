# Color de cucardas

Ese sub-módulo permite autoadministrar el color de las cucardas maquetadas por la marca. Los colores administrables serán los de fondo y texto.

{% hint style="info" %}
Para utilizar la funcionalidad se deberán realizar una modificación en el maquetado de las cucardas. Deberás solicitarlo mediante [Formulario](https://form.asana.com/?k=v_sPm4Y17NxYHDezOEKt8A\&d=66032875349265).
{% endhint %}

Se podrán administrar los colores de fondo y texto de las cucardas generales como:

* Cucarda porcentaje de descuento
* Cucarda Custom (1)
* Cucarda Custom Bis (2)
* Cucarda Sale
* Cucarda New in

### Configuración <a href="#configuracion" id="configuracion"></a>

1. Buscar la cucarda que se quiere modificar y asignar un color tanto de fondo como de texto:

Dicho color se podrá elegir tanto:

* Desplegando el cursor por el panel de color
* Desplegando el cursor por la barra de color
* Completando los campos R G y B (como se muestra en la imagen de ejemplo)
* Completando los campos H S y L
* Completando con formato hexadecimal
