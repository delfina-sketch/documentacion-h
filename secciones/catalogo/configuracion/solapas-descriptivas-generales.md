# Solapas descriptivas generales

### Descripción <a href="#descripcion" id="descripcion"></a>

Esta herramienta permite auto-administrar los textos de las pestañas descriptivas para todos los productos de la web. Las mismas son:

* Métodos de envío
* Métodos de pago
* Cambios y devoluciones

Se podrán configurar hasta 3 pestañas con Título y Texto (mismo contenido para todos los productos) que se adaptarán según el diseño que se tenga en la web.

Importante: si la solapa contiene banners, los mismos se mantendrán y solo se podrá modificar el texto.

### Configuración <a href="#configuracion" id="configuracion"></a>

1. Completar los campos según la información que quiera mostrar en la solapa:

<figure><img src="../../../.gitbook/assets/image (44).png" alt=""><figcaption></figcaption></figure>

* Título: el texto se colocará como nombre de la solapa
* Texto: al desplegar la solapa se visualizará la información correspondiente al título desplegado.

Para separar párrafos, en el back se debe colocar \<br>. Por ejemplo, si se coloca \<br> luego de "No válido para compras de Outlet", y nuevamente \<br>, se verá el texto separado por párrafos con un espacio:
