# Configuración

**URL:** `/admin/catalogo/configuracion`

En este módulo se podrán configurar opciones visuales y de comportamiento del catálogo en la tienda como:&#x20;

| Tab                      | Descripcion                                                    |
| ------------------------ | -------------------------------------------------------------- |
| **Out of Stock**         | Funcionalidad para productos sin stock                         |
| **Cucardas**             | Badges/etiquetas visuales para productos (ej: "Nuevo", "Sale") |
| **Cucardas PNG**         | Cucardas personalizadas con imágenes PNG subidas               |
| **Solapas Descriptivas** | Tabs adicionales en la pagina de detalle de producto           |
| **Cucarda Envio Gratis** | Cucarda especifica para productos con envío gratis             |
